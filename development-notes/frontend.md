# Frontend

## Para rodar o servidorzinho do projeto
```bash
npm run dev
```

O servidor geralmente abre na [porta 5173](http://localhost:5173/).

## Entendendo o Vite + React

### `src/main.tsx`
É onde fica o esqueleto da aplicação: o `<App />`.

###  `src/app/App.tsx`
É onde renderizamos a Home do projeto -- que também é o projeto inteiro.
